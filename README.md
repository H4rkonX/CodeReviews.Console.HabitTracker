# Habit Tracker

A C# application that performs CRUD operations using SQLite, allowing users to track, update, and delete their habits.

## Features

- **Interactive Menu** — insert, update, view, and report habits.  
  ![Menu](https://github.com/H4rkonX/Console.HabitTracker/blob/main/.assets/menu.png)

- **View Recorded Habits** — display all saved habits in a styled table.  
  ![View all records](https://github.com/H4rkonX/Console.HabitTracker/blob/main/.assets/viewrecords.png)

- **Habit Report** — visualize details of a selected habit in a friendly format.  
  ![Habit information](https://github.com/H4rkonX/Console.HabitTracker/blob/main/.assets/habitreport.png)

# Packages used
- [Spectre.Console](https://spectreconsole.net)
- [SQLite](https://www.sqlite.org/index.html)
